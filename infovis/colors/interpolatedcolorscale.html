Interpolated Color Scale

A continuous color scale is used to represent quantitative continuous values. For values \(u \in [u_{min}, u_{max}]\) a mapped color can obtained by linearly interpolating the color value from a color lookup table. Certain applications might require a special interpolation scheme, such us logarithmic scale.

It is possible to choose different color scales. By moving the mouse over the color scale one can see the corresponding hex-code and rgb values of the colors.
