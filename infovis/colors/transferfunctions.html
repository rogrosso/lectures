Transfer Functions

In this example, the transfer functions for a color scale are defined for a scalar field, taking values in the interval \([0,1]\). The scalar value might represent any quantitative data, such as a temperature distribution. Four transfer functions have to be defined; the fourth one is transparency. For example, this plays a crucial role in applications in the scientific visualization of volume data.
