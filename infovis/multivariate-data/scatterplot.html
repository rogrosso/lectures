Scatter Plot

A scatter plot is a graph representation of pairs of values commonly in Cartesian coordinates. The pattern resulting by plotting a data set might reveal any correlation between the variables being measured. Scatter plots are appropriated for visualizing the association/correlation between two quantitative variables.

The scatter plot below provides some interaction. The data set being visualized are auto-mpg used in 1983 for statistical purposes, and the Iris flower data set introduced in 1936 by Roland Fischer for use in taxonomy problems.
