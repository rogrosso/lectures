Lectures

Roberto Grosso

Project start: Feb. 2023

This repository contains the HTML and Javascript code implementing algorithms discussed in some of the lectures that I give at the University. They are in the area of geometry processing and information visualization. This way, the code and some background information about the algorithms can be freely accessed not only from inside the University.

You will find a lot of duplicated code. I hope this will become better in the future. The web applications are interactive. I am using D3 for 2D graphics and three.js for surfaces. I strongly recommend to interact with the visualizations by selecting and dragging points or by zooming and rotating the 3D geometry. Some of the applications also include a kind of user interface. The code is at this moment certainly buggy. The code was not implemented for industrial usage.

Example: Catmull-Clark Subdivision Surface

Remark

It is recommended to start with the file index.html in the root/main folder in order to easily navigate through all the applications implemented for the different lectures. In order to run the code, a live-server is required. I personally use Visual Studio Code and the live-server plugin. I have been having problems with the CDNs, in particular unpkg with D3. If a web application does not work, please, just edit the html file and comment in other CDN.
